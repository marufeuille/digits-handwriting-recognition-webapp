# Usage

0. pre requirement
- install keras, tensorflow(CPU), scikit-learn

```bash
pip install keras tensorflow tensorflow-cpu scikit-learn
```

1. clone this repo

```bash
git clone https://github.com/marufeuille/digits-handwriting-recognition-webapp
```

2. Run app

```bash
python3 app.py
```

3. access to 8080 ports

## external library

this app uses [DrawingBoard](https://github.com/Leimi/drawingboard.js)
